Welcome to ELEA!

With ELEA you can build your own Evolutionary Algorithm with blocks. ELEA uses Blockly to define blocks and their functionality. Take a look at examples or build your Algorithm with the Initialize Programm block, define your score function and run your program. If you create an algorithm from an empty starting point, please be sure to define a function to named fitness with your score function.

Check out this while-loop if you want to know how to use ELEA with the IOHanalyzer.

On this page, every block you can use in ELEA is documented. The documentation is separated into the categories you can also find in the workspace:

  • Population
  • Individuals
  • Fitness
  • Primitive datatypes
  • Logic
  • Loops
  • Functions
  • Logging
  • Timer management
  • Multi-Threading
  • Experimental

INFO Please be aware that some of the blocks you can find in the workspace are predefined by blockly and not documented here yet.

Population

In the Population-Category, you find methods to transform, merge and create populations and select individuals.

Individuals

In the Individuals-Category, you find methods to mutate individuals and crossover them.

Fitness

In the Fitness-Category, you find methods that help you to define a score function.

Primitive datatypes

In the Primitive-datatypes-Category, you find methods that help you to handle primitive datatypes like Strings and Integers.

Logic

In the Logic-Category, you find logical methods to compaire numbers, if-else -statements or logical operation. Every method is predefined by Blockly.

Loops

In the Loop-Category, you can find for -, for-each - and while -loops. Every method is predefined by Blockly.

Functions

In the Functions-Category, you can create your functions. After initializing them, you can find a block named after your function, which you can use in your algorithm.

INFO: If you create an algorithm from an empty starting point, please be sure to define a function to named fitness with your score function.

Logging

In the Logging-Category, you find methods to print to the output, show diagrams, save to CSV and add comment-blocks.

Multi-Threading

In the Multi-Threading-Category, you find methods that help you use multi-threading in ELEA.

Timer management

In the Time-Category, you find methods to track execution-time.
