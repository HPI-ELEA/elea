What are Evolutionary Algorithms?

Evolutionary Algorithms are algorithms that try to optimize a given objective function heuristically. As the name suggests, the inspiration comes from nature, and the algorithm maintains a set of possible solutions (then called ‚Äúindividuals‚Äù) and evaluates their objective value (called ‚Äúfitness‚Äù).
